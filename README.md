# 🎥 Live Object Detection & Tracing App

A real-time object detection web application built with Streamlit, WebRTC, and YOLOv8.  
This app uses your webcam to detect and track objects live in the browser.

---

## 🚀 Features

- 📷 Real-time webcam object detection
- 🎯 YOLOv8 object tracking
- ⚡ Fast live processing
- 🌐 Streamlit web interface
- 🧠 AI-powered object recognition

---
